# Eidosc Tools 0.6.0-alpha.1 dot Namespace support

- Updated Eidos and Eidosc compatibility metadata to `>=0.6.0-alpha.1 <0.7.0`.
- Reserved `::` highlighting for name-first declaration bindings and removed legacy qualified-name and slash-path grammar rules.
- Added dot Namespace highlighting for uppercase module roots and lowercase dependency aliases followed by uppercase modules.
- Updated completion word ranges and lexical semantic-token fallback to treat `.` as the single Namespace selector without confusing runtime member chains.
- Kept ADT constructor commas under the standard separator scope and removed `|`-separator assumptions from the editor contract.
