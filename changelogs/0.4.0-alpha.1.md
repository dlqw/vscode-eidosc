# 0.4.0-alpha.1

- Replaced ability declaration scopes with nominal effect-tag scopes.
- Added highlighting for `effect`, `need`, and the `effects` row kind.
- Removed handler, `with`, `resume`, and ability keywords from lexical and semantic editor contracts.
- Renamed the semantic token type to `effectTag` while retaining the `effect` occurrence modifier.
- Made `package.json` the authoritative plugin version source.
- Updated manifest support to `manifestSchema = 3` and `[language].version`.
